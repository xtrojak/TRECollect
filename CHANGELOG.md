# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.1.0] - 2026-02-26

### Added

- general support of sites creation and their overview [#1](https://github.com/grp-bork/TRECollect/issues/1)
- support for sampling sites depending on the selected sampling team [#5](https://github.com/grp-bork/TRECollect/issues/5)
- local and remote submission storing [#3](https://github.com/grp-bork/TRECollect/issues/3)
- various widget types: location, date, time, text, single and multi select, photo, barcode scanner, tabular input, and dynamic widget [#2](https://github.com/grp-bork/TRECollect/issues/2)
- other initial functionaly as described in [#29](https://github.com/grp-bork/TRECollect/pull/29)
